Sports Widget for Plasma icon assets
====================================

The bundled sport and stadium SVG icons are sourced from Iconify collections:

- Material Symbols: football, basketball, american-football, baseball, tennis, hockey, cricket, volleyball, stadium
- Sports Widget for Plasma: snooker

Material Symbols is licensed under Apache-2.0. A copy of the Apache-2.0
license is included in this directory as APACHE-2.0.txt. The snooker icon is
part of this project and follows the project license.
